# Scream

Scream is a fast screensharing app for macOS devices. It uses macOS APIs (`ScreenCaptureKit`) to capture the screen

### Roadmap
- Screen recording
- Streaming
- Authentication
- Client app
- Server app
- iOS Client app

### Compilation
Make sure Xcode is installedd
Clone the repo
Open the project in Xcode
Select your Team ID
Set the destination to "My Mac"
Hit Command + R
Make sure to allow screen recording permissions
